[しと鯖](https://mc.citringo.net/) [modpackage API v1](http://api.citringo.net/citserver/modpackage/v1) のクライアント

## 動作環境
- Python 2系

## 使い方
`url.json`を変更してダウンロード先を変更してから`python downloader.py`するだけ  
実行ディレクトリに保存されます  
保存先変更はまだ**対応してない**です  
差分更新はまだ**対応してない**です
